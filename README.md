# Где кадры, Зин?

Книга для собственника о людях: как нанимать, платить, учить и расставаться так, чтобы дело не держалось на одном незаменимом герое.

Живая книга, дописывается глава за главой. Примеры из мебельного дела, логика для любого бизнеса.

**Читать:** https://abdublinn.github.io/gde-kadry-zin/

Автор: Александр Дублин · [Дзен](https://dzen.ru/dbs_dubas) · [Телеграм](https://t.me/ab_dublin)
